## Запуск

### Общее

Запускать с аргументами командной строки:

1. Путь до файла с данными (который был `u.data`)
2. Id фильма
3. Путь, куда писать результат

### Sbt

Я запускал через Sbt.

``` sh
> sbt
> run <path_to_data> <film_id> <out>
```
